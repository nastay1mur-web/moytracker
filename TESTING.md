## 2. Локальный запуск

### Вариант A: открыть напрямую в браузере (быстрый)

```bash
# Windows
start tg-app/index.html

# macOS / Linux
open tg-app/index.html
```

Файл открывается как `file://...` — этого достаточно для проверки большинства функций.

**Ограничение:** `window.Telegram.WebApp` будет `undefined` — это ожидаемо. Код это обрабатывает: `const tg = window.Telegram?.WebApp;`. Все функции кроме `tg.openTelegramLink()` будут работать.

### Вариант B: через локальный HTTP-сервер (точнее)

Нужен любой статический сервер. Если установлен Node.js:

```bash
npx serve tg-app
# открыть http://localhost:3000
```

Или Python:

```bash
cd tg-app
python -m http.server 8080
# открыть http://localhost:8080
```

### Вариант C: через Telegram (полная проверка)

1. Открыть Telegram
2. Найти бота `@tracker_privichka_bot`
3. Нажать кнопку "Menu" или отправить `/start`
4. Mini App откроется на `https://tg-app-green.vercel.app`

## 3. Деплой на Vercel

**Критически важно:** проект связан с Vercel через CLI, а НЕ через GitHub auto-deploy. Пуш в GitHub НЕ обновляет продакшн автоматически.

```bash

npx vercel --prod --scope nastay1mur-webs-projects
```

После деплоя проверить, что `https://tg-app-green.vercel.app` обновился (можно проверить через хард-рефреш `Ctrl+Shift+R`).

## 5. Структура данных (localStorage)

Приложение хранит всё в `localStorage`. Ключи:

| Ключ | Тип | Описание |
|---|---|---|
| `tg_habits` | `Array<Habit>` | Список привычек |
| `tg_completions` | `Array<Completion>` | История выполнений |
| `tg_settings` | `Settings` | Настройки пользователя |
| `offerShown` | `boolean` | Показывался ли экран-оффер |

**Схема `Habit`:**
```json
{
  "id": 1,
  "emoji": "💧",
  "name": "Выпить 8 стаканов воды",
  "category": "Здоровье",
  "reminders": ["08:00", "12:00"]
}
```

**Схема `Completion`:**
```json
{ "habitId": 1, "date": "2026-03-07" }
```

**Схема `Settings`:**
```json
{
  "notifications": true,
  "morningTime": "08:00",
  "eveningTime": "21:00",
  "theme": "Системная",
  "weekStart": "Понедельник",
  "hideFailures": false
}
```

**Привычки по умолчанию** (загружаются при первом старте, если `tg_habits` пустой):
- 💧 Выпить 8 стаканов воды — Здоровье
- 🧘 Медитация 10 минут — Осознанность
- 🤸 Зарядка / Растяжка — Фитнес
- 📖 Читать 20 страниц — Развитие
- 🌙 Ложиться спать до 23:00 — Сон
- 🌳 Прогулка 30 минут — Активность

### TC-16: Расчёт streak (серии дней)

**Предусловие:** занести в `localStorage` вручную данные для проверки

Через DevTools (F12 → Console):
```js
// Симуляция выполнения вчера и позавчера для привычки id=1
const today = new Date();
const yesterday = new Date(today); yesterday.setDate(today.getDate()-1);
const dayBefore = new Date(today); dayBefore.setDate(today.getDate()-2);

localStorage.setItem('tg_completions', JSON.stringify([
  { habitId: 1, date: yesterday.toISOString().slice(0,10) },
  { habitId: 1, date: dayBefore.toISOString().slice(0,10) }
]));
location.reload();
```

**Ожидаемый результат:**
- Для привычки id=1 (Выпить 8 стаканов воды) streak = 2
- Алгоритм: считает подряд начиная со вчера (если не выполнено сегодня) или сегодня (если выполнено)

## 7. Сброс состояния для тестов

### Полный сброс (первый запуск):
```js
// В DevTools Console (F12)
localStorage.clear();
location.reload();
```
После этого: оффер покажется снова, привычки — 6 дефолтных, никаких выполнений.

### Сброс только оффера:
```js
localStorage.removeItem('offerShown');
location.reload();
```

### Сброс только выполнений:
```js
localStorage.removeItem('tg_completions');
location.reload();
```

### Просмотр всех данных:
```js
// Все ключи
['tg_habits','tg_completions','tg_settings','offerShown'].forEach(k =>
  console.log(k, JSON.parse(localStorage.getItem(k)))
);
```

## 8. Известные ограничения (не баги)

| Поведение | Причина |
|---|---|
| Напоминания в Telegram не приходят | Бэкенд не реализован. Времена сохраняются в localStorage для будущей интеграции |
| `setTime()` использует `prompt()` вместо нативного picker | Временное решение. Принимает только формат `ЧЧ:ММ` |
| Данные не синхронизируются между устройствами | Нет бэкенда. Данные только в localStorage конкретного браузера/устройства |
| `window.Telegram.WebApp` === undefined в браузере | Штатное поведение. Код обрабатывает через `?.` опциональную цепочку |
| В настройках нельзя изменить тему и начало недели | UI-заглушки для будущих функций |

## 10. Структура кода (tg-app/index.html)

Весь JavaScript обёрнут в IIFE `(function() { ... })()`. Глобально доступен только объект `window.App`.

| Функция/метод | Что делает |
|---|---|
| `todayStr()` | Возвращает текущую дату в формате `YYYY-MM-DD` |
| `load(key, fallback)` | Читает и парсит JSON из localStorage |
| `save(key, value)` | Сериализует и сохраняет в localStorage |
| `fmt12(time)` | `"08:00"` → `"8:00"` (убирает ведущий ноль) |
| `isDoneToday(id)` | Проверяет, выполнена ли привычка сегодня |
| `getStreak(id)` | Считает серию подряд идущих дней (макс. 365) |
| `getWeeklyPct()` | Процент за последние 7 дней, с учётом hideFailures |
| `getBestDay()` | Лучший день недели по количеству выполнений |
| `htmlToday()` | Рендер вкладки "Сегодня" |
| `htmlProgress()` | Рендер вкладки "Прогресс" |
| `htmlSettings()` | Рендер вкладки "Настройки" |
| `htmlRemindersModal(id)` | Рендер модала напоминаний для привычки |
| `htmlAddModal()` | Рендер модала добавления привычки |
| `htmlOffer()` | Рендер экрана-оффера |
| `render()` | Полная перерисовка `#root` |
| `App.tab(name)` | Переключение активной вкладки |
| `App.complete(id)` | Отметить привычку выполненной (частичное обновление DOM) |
| `App.del(id)` | Удалить привычку с подтверждением |
| `App.openReminders(id)` / `closeReminders(e)` | Открыть/закрыть модал напоминаний |
| `App.addReminder(id)` / `removeReminder(id, time)` | Добавить/удалить время напоминания |
| `App.saveReminders(id)` | Сохранить напоминания и обновить метку на карточке |
| `App.openAdd()` / `closeAdd(e)` | Открыть/закрыть модал добавления |
| `App.pickEmoji(e)` / `pickCat(c)` | Выбор emoji и категории в модале |
| `App.saveHabit()` | Сохранить новую привычку |
| `App.toggleHideFailures()` | Переключить режим скрытия поражений |
| `App.toggleNotif()` | Переключить уведомления |
| `App.setTime(type)` | Изменить дефолтное время через prompt |
| `App.clearAll()` | Сбросить все данные к умолчаниям |
| `App.openBot()` | Открыть бота в Telegram |
| `App.closeOffer()` | Закрыть оффер и записать флаг |
