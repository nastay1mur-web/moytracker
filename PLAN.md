### Готовый файл: MAX-app/offer-tracker.html

Одностраничное Mini App с экраном-оффером. Файл полностью готов к деплою на Vercel.

**Что умеет:**
- Экран-оффер при первом открытии (показывается один раз, флаг `offerShown` в localStorage)
- 6 привычек по умолчанию с кнопками «Готово»
- Счётчик серий (streak) на каждой привычке
- Прогресс-бар дня «Выполнено: X/6»
- Тёмная тема через `prefers-color-scheme`
- Совместимость с Telegram/MAX WebApp SDK

**Что нужно изменить перед запуском:**
- Заменить `{username_бота}` на реальный username бота (строка `const BOT_USERNAME`)
- При необходимости — текст оффера и список привычек

**Технические детали экрана-оффера:**

| Элемент | Значение |
|---------|---------|
| Триггер показа | `localStorage.getItem('offerShown')` → null = показать |
| Закрытие | Клик «Пропустить» → `offerShown = 'true'` → renderTracker() |
| Кнопка подписки | `https://t.me/{username}?start=from_app` |
| Анимация появления | `fadeIn` 0.3s + `slideUp` 0.3s |
| Фон | `rgba(0,0,0,0.5)` |
| Карточка | белая, border-radius 24px, padding 24px |
| CTA-кнопка | `#2AABEE`, min-height 48px, border-radius 16px |

**Структура привычек по умолчанию:**
```
💧 Выпить 8 стаканов воды   — Здоровье
🧘 Медитация 10 минут       — Осознанность
🤸 Зарядка / Растяжка       — Фитнес
📖 Читать 20 страниц        — Развитие
🌙 Ложиться спать до 23:00  — Сон
🌳 Прогулка 30 минут        — Активность
```

## Ключевые отличия Telegram от MAX

| Параметр | MAX | Telegram |
|---|---|---|
| Регистрация бота | Только юрлицо / ИП | Любой через @BotFather |
| Аудитория | ~Россия/СНГ | 900M+ по всему миру |
| SDK | Совместим с Telegram | Оригинальный |
| Монетизация | Telegram Stars / реклама | Telegram Stars / Stars / донаты |
| Готовый прототип | Есть (MAX-app/) | Нет (начинаем) |

## Заметки

- Готовый прототип MAX-app/ можно переиспользовать для Telegram-версии:
  MAX Web App SDK полностью совместим с `window.Telegram.WebApp` — замена одного файла.
- Основная бизнес-логика (storage.js, app.js, app.css) переносится без изменений.
- Главное отличие Telegram-версии: нужен реальный бэкенд (не localStorage) для синхронизации данных между устройствами.
